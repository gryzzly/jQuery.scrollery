Simple and light-weight plugin that allows to create a film-like gallery from a list of images, wrapped in a scrollable container.

Click on the container in order to continue to the next image. When the last image is reached, gallery scrolls to the first item.

The main benefit of using this gallery is that it's width-independent. What it means is that the container can have flexible width.
